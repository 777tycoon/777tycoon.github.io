git bash 基本指令 from [教學] git-4 bash基本指令操作 - 討論區  |   NVDA 台灣

在 git bash 中，路徑都是以 / 正斜線來隔開，如果是 D 槽則是以 /d 來表示，

這稱為絕對路徑，就是從最前面的根目錄開始寫起，/ 就是代表根目錄。

第二行，也就是游標所在行，前面有一個錢號，這是 git bash 的指令列提示符號，在輸入指令時不需輸入該符號喔。
