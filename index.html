拾貝投資讀書會：一起探索財務自由的道路

緣起

原本希望是十倍成長或是十倍速，但後來我發現欲速則不達，所以選擇要慢一點，才能夠彎腰撿到寶貝，所以稱為拾貝。

歡迎加入我們的投資讀書會！這裡是一個為熱愛學習與成長的朋友打造的交流平台，透過分享與討論，攜手提升投資知識與技能，共同實現財務目標。

我們的宗旨

  1. 學習成長：發掘投資盲點，學習專業知識，拓展市場觀點。
  2. 創造機會：探索新商機與創意，發現財務自由的可能性。
  3. 互相支持：建立互助網絡，分享資源，解決挑戰。

運作模式

  • 線上交流：利用 Line 群組與 Google Meet，隨時隨地參與討論與學習。
  • 每月主題：由成員輪值領讀，探討熱門投資話題或經典書籍。
  • 實體聚會：每季一次，面對面深入交流

我們適合誰？

  • 熱衷於主動投資，想提升權益數的你。
  • 對股票、期貨或產業趨勢感興趣，希望學習新知識的你。
  • 願意分享經驗，並一起面對投資挑戰的你。

為什麼加入我們？

這不僅僅是一個讀書會，更是一個支持彼此實現夢想的團體。一起激盪思維，讓每一次討論成為改變未來的契機！

立即行動！
如果你對我們的讀書會感興趣，歡迎聯絡我們或直接留言加入！

在 Hexo 中使用 NexT 佈景主題 設置 Menu（選單） 可以讓你的網站更具結構性。以下是完整的設置步驟：

🛠 步驟 1: 打開 NexT 配置文件

  1. 前往你的 Hexo 根目錄。
  2. 打開 NexT 主題配置文件，通常位於：

📝 步驟 2: 編輯 Menu 設定

找到配置文件中的 menu 區塊，通常看起來像這樣：

🧩 Menu 配置詳解

每一行都有三個部分：

  • **menu_item**：選單項目的名稱（例如 home, about 等）。
  • **路徑**：指向的 URL 路徑（例如 /about/）。
    • 根目錄頁面使用 /。
    • 其他頁面使用 /page-name/。
  • **圖標**：使用 Font Awesome 圖標（可選）。
    • 例如：fa fa-home 代表首頁圖標。

📌 小提示

  • Font Awesome 圖標：可以在 Font Awesome 網站 查找更多圖標。
  • 順序調整：修改配置文件中選單項目的順序會影響它們在網站上的排列順序。

改完發現還沒出來，原來是預設的 config.yml 設定中，menu 部分的項目都被註解（#）掉了，這就是為什麼 Menu bar 沒有顯示的原因。取消註解（去掉 # 符號）讓選單項目就顯示出來了。

🛠️ 修正方法：取消註解

將以下內容中的 # 刪除，保持每行的格式正確，這樣它們才會在網站中顯示：

yaml

複製程式碼

menu: home: / || fa fa-home about: /about/ || fa fa-user tags: /tags/ || fa fa-tags categories: /categories/ || fa fa-th archives: /archives/ || fa fa-archive # 如果不需要某項，可以保留註解或刪除該行

🔍 檢查格式：

  1. 縮排：保持 menu: 下的每個項目都有兩個空格縮排。
  2. 連結設定：
    • home: / 表示首頁連結。
    • /about/ 是指向你的「關於」頁面。請確認你的 Hexo 中是否有這些頁面（例如 source/about/index.md）。
  3. 圖示設定：|| fa fa-home 使用的是 Font Awesome 圖標。如果你沒有引入 Font Awesome，也可以移除 || 後的圖標部分。

在 git bash 中，路徑都是以 / 正斜線來隔開，如果是 D 槽則是以 /d 來表示，
